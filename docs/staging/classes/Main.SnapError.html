    Class SnapError

    A generic error which can be thrown by a Snap, without it causing the Snap to crash.

    • new SnapError(
          error: string | Error | JsonRpcError,
          data?: Record<string, Json>,
      ): SnapError

      Create a new SnapError.

      Parameters

      • error: string | Error | JsonRpcError

        The error to create the SnapError from. If this is a string, it will be used as the error message. If this is an Error, its message property will be used as the error message. If this is a JsonRpcError, its message property will be used as the error message and its code property will be used as the error code. Otherwise, the error will be converted to a string and used as the error message.

      • data: Record<string, Json> = {}

        Additional data to include in the error. This will be merged with the error data, if any.

      Returns SnapError
